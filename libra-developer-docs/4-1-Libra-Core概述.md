# Libra Core概述
Libra Core是Libra协会发布的基于Libra protocol的开源实现的官方名称。

- 该软件是Libra协议和Move语言的第一个实现
- Libra Core包括验证程序和客户端功能。
- Libra Core是用Rust写的。
